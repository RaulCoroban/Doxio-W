Take control and keep your private stuff really private with the private Vault Gallery.

With an intuitive interface, the app allows you to store full folders containing media in a safe place, requiring a PIN code or any other kind of security measure to access them. Besides, your files will be backed up automatically so you can have them in any device.

All images in this concept come from a free stock source. All icons have been designed specifically for this app.
